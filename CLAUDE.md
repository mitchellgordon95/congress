# Congress Chat — Repo Root

This repo is two things: the Congress Chat codebase (in `code/`) and a knowledge base that serves as Mitchell's thought partner on the project.

**Read `AGENTS.md` first.** It is the agent entry point: standing rules, attribution conventions, and how the wiki/sources/log system works.

## Layout

- `code/` — the app. **All build commands run from `code/`.** Technical docs in `code/CLAUDE.md`.
- `AGENTS.md` — agent entry point and standing rules.
- `wiki/` — agent-maintained knowledge base (`wiki/main.md` is the master article).
- `sources/` — source cards for external material.
- `log/` — append-only weekly activity logs.
- `meetings/` — interviews and call notes.
- `docs/` — longform internal docs (incl. `ui_mocks.txt`).
- `vision.md` — vision and strategy.
